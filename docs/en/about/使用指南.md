# User Guide

## Reading Articles

### Navigation Categories

Each article is categorized and has a navigation bar. Hover over a module to see its subcategories, and use the navigation bar to quickly locate the relevant knowledge points you want to study.

![](https://ckcsec.oss-cn-hangzhou.aliyuncs.com/img/image-20240619170548266.png)

### Code Highlighting

Articles involving code are configured with syntax highlighting, giving researchers a better reading experience.

![](https://ckcsec.oss-cn-hangzhou.aliyuncs.com/img/image-20240619170937497.png)

### Content Search

The library includes content search so you can quickly find the articles you want to read.

![](https://ckcsec.oss-cn-hangzhou.aliyuncs.com/img/image-20240619171111699.png)

## Partners

If your project or company wants more exposure and wants more security researchers to know about it,

you can package your project in `Markdown` format and send it to `admin@ckcsec.com`.

After review and approval, it will appear in the friend links section.
