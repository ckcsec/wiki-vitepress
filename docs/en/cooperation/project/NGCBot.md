# ngc660sec/NGCBot

![](https://ckcsec.oss-cn-hangzhou.aliyuncs.com/img/Logo2.png)

## Project Introduction

A WeChat bot based on the ✨HOOK mechanism. It supports scheduled 🌱security news pushes from FreeBuf, Xianzhi, Anquanke, and the QiAnXin Attack and Defense Community; 👯KFC copywriting; ⚡ICP filing lookup; ⚡phone number attribution lookup; ⚡WHOIS lookup; 🎉horoscope lookup; ⚡weather lookup; 🌱slacking-off calendar; ⚡ThreatBook threat intelligence lookup;
🐛beauty videos; ⚡beauty images; and a 👯help menu. 📫 It supports a points system, ⚡automatic group invites, ⚡advertisement detection, 🌱automatic mass messaging, 👯AI replies, and rich customization, making it easy for beginners to get started!
